# Contributing to DS Mini Projects

Thank you for your interest in contributing!
This guide explains how anyone can contribute to this repo.

---

## What is Contributing?

Contributing means helping improve someone else's project by:
- Fixing a bug or mistake
- Adding a new feature or project
- Improving documentation
- Suggesting ideas via Issues

---

## Step by Step: How to Contribute

### Step 1 - Fork the repo
Click the Fork button on the top right of this page.
This creates YOUR personal copy of this repo.

### Step 2 - Clone your fork
```
git clone https://github.com/YOUR-USERNAME/DS-Mini-Projects.git
cd DS-Mini-Projects
```

### Step 3 - Create a new branch
Never work directly on main. Create a branch for your change:
```
git checkout -b feature/your-feature-name
```

### Step 4 - Make your changes
Add your mini project, fix a bug, or improve docs.

### Step 5 - Commit your changes
```
git add .
git commit -m 'feat: add your descriptive message here'
```

### Step 6 - Push to your fork
```
git push origin feature/your-feature-name
```

### Step 7 - Open a Pull Request
Go to the original repo on GitHub.
Click Compare and pull request.
Describe your changes clearly and submit!

## What Can You Contribute?

| Type | Example |
|---|---|
| New mini project | A weather analyser, quiz game, unit converter |
| Bug fix | Fixing a broken code cell |
| Docs improvement | Clearer explanation, better comments |
| New exercise | Add practice questions to existing projects |

---

## Commit Message Format

Use clear prefixes so everyone understands the change:

| Prefix | When to use |
|---|---|
| feat: | Adding something new |
| fix: | Fixing a bug |
| docs: | Updating documentation |
| chore: | Cleanup or maintenance |

Example: feat: add temperature converter mini project

---

## Raising an Issue

Found a bug or have an idea? Great!
Go to the Issues tab and click New Issue.
Describe it clearly - what you found, what you expected.

---

## Code of Conduct

Be kind. Be constructive. Be helpful.
This is a learning space - all levels welcome!
